要判断两个矩形是否相交，我们发现会有很多种情况。

要依次分析会很麻烦，而且容易出错。

所以，从反面看问题比较简单。先判断什么情况下两个矩形不会相交。

与之相反时，两个矩形相交。

这里使用： element.getBoundingClientRect();

两个矩形不相交时，有四种情况，

1. 当 RectB 在 RectA 左边
RectB. right < RectA. left

2. 当 RectB 在 RectA 右边

RectB. left > RectA.right

3. 当 RectB 在 RectA 上面
RectB. bottom < RectA. top

4. 当 RectB 在 RectA 下面
RectB. top > RectA. bottom


function checkIntersect(RectA,RectB)
{

  nonIntersect =

 （ RectB.right < RectA.left )||

 （ RectB.left > RectA.right ）||

 （ RectB.bottom < RectA.top ）||

 （ RectB.top > RectA.bottom ）；

      //相交
  intersect = !nonIntersect;

}
